# MEESHO WOMEN'S FASHION DATABASE MANAGEMENT SYSTEM

## Overview

This repository contains the database design and requirement analysis for the **Meesho Women's Fashion Database Management System**. The project focuses on designing a relational database to manage products, customers, sellers, orders, payments, and delivery information efficiently.

## Problem Statement
The rapid growth of e-commerce has increased the need for an efficient database system to manage customer information, women's fashion products, sellers, orders, payments, and delivery records. Managing these operations manually can lead to duplicate records, inaccurate product information, delayed order processing, inventory issues, and difficulty in retrieving data.

To overcome these challenges, this project proposes a structured relational database that organizes business data efficiently, minimizes redundancy, maintains data integrity, and supports the daily operations of the Meesho Women's Fashion platform.

## Project Description

The database manages:

* Customer Management
* Seller Management
* Product Management
* Category Management
* Order Management
* Payment Management
* Delivery Management

The project uses DBMS concepts, ER modeling, normalization, and SQL to build an efficient and reliable database.
